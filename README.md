# Cyberpersona-lens

CyberPersona Lens is a Flask-based cybersecurity web application that helps users explore their digital exposure and understand their potential cybersecurity risk profile.

---

## Overview

CyberPersona Lens allows users to enter an email address and analyse possible digital exposure. The application simulates breach awareness features and generates a simple cyber persona summary that highlights potential security risks.

This project focuses on improving awareness of personal cybersecurity and digital footprint exposure.

---

## Features

• Email-based exposure analysis  
• Simulated breach awareness results  
• Risk score calculation  
• Cyber persona summary generation  
• Interactive web interface

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/AudreyShelly3/cyberpersona-lens.git
```

2. Navigate to the project folder

```
cd cyberpersona-lens
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the Flask application

```
python app.py
```

5. Open in your browser

```
http://127.0.0.1:5000
```
